  cpu-z80 - Changelog
================================================================================

v0.1 (2026-04-01)
------------------
Prvni vydani presneho Z80A emulatoru.

Instrukcni sada:
  - Kompletni oficialni instrukcni sada Z80A (245 instrukci)
  - CB prefix: RLC, RRC, RL, RR, SLA, SRA, SRL, BIT, SET, RES
  - ED prefix: block presuny (LDI/LDIR/LDD/LDDR), block porovnani
    (CPI/CPIR/CPD/CPDR), block I/O (INI/INIR/IND/INDR/OUTI/OTIR/OUTD/OTDR),
    IN r,(C), OUT (C),r, ADC/SBC HL, LD (nn),rr, NEG, RETN, RETI, IM, RRD, RLD
  - DD/FD prefix: kompletni IX/IY podpora vcetne (IX+d) adresovani
  - DD CB/FD CB prefix: indexed bit operace s kopirovani do registru
  - Nedokumentovane instrukce: IXH, IXL, IYH, IYL, SLL (CB 30-37)

Presnost:
  - Presne pocitani T-stavu pro kazdou instrukci
  - Spravne chovani vsech flagu vcetne nedokumentovanych bitu F3 (bit 3)
    a F5 (bit 5)
  - Interni registr MEMPTR/WZ s korektnim vlivem na F3/F5
  - DAA s kompletni BCD korekcni logikou (2048-entry lookup tabulka)
  - LD A,I/R HW bug: INT po LD A,I nebo LD A,R resetuje PF na 0

Prerusovaci system:
  - IM0: vektorove preruseni (instrukce na datove sbernici)
  - IM1: pevny skok na 0x0038
  - IM2: vektorove preruseni (I:vector -> tabulka)
  - NMI: nemaskovane preruseni (skok na 0x0066), zachovava IFF2
  - EI delay: po EI se preruseni odlozi o jednu instrukci
  - INTACK callback pro daisy chain periferií
  - RETI callback pro notifikaci periferií
  - Level-sensitive INT linka (polling callback)

Optimalizace:
  - Computed goto dispatch (GCC/Clang)
  - Lokalni registrova cache v z80_execute()
  - Eliminace null-checku callbacku (default handlery)
  - DAA lookup tabulka
  - Inline prefix handlery

API:
  - z80_init(), z80_reset()
  - z80_step(), z80_execute()
  - z80_irq(), z80_nmi()
  - Callback system pro pamet, I/O, preruseni
  - z80_add_wait_states() pro extra cekaci stavy z callbacku
  - z80_set_post_step_fn() pro per-instrukce callback

Testovaci sada:
  - 402 testu pokryvajicich vsechny instrukce, flagy, casovani,
    preruseni, MEMPTR a nedokumentovane chovani
  - ZEXALL validace: 67/67 testu Z80 Instruction Exerciser (vcetne
    nedokumentovanych instrukci a flagu)
