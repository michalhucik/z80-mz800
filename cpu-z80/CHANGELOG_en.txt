  cpu-z80 - Changelog

v0.1 (2026-04-01)
------------------
First release of the accurate Z80A emulator.

Instruction set:
  - Complete official Z80A instruction set (245 instructions)
  - CB prefix: RLC, RRC, RL, RR, SLA, SRA, SRL, BIT, SET, RES
  - ED prefix: block transfers (LDI/LDIR/LDD/LDDR), block compare
    (CPI/CPIR/CPD/CPDR), block I/O (INI/INIR/IND/INDR/OUTI/OTIR/OUTD/OTDR),
    IN r,(C), OUT (C),r, ADC/SBC HL, LD (nn),rr, NEG, RETN, RETI, IM, RRD, RLD
  - DD/FD prefix: complete IX/IY support including (IX+d) addressing
  - DD CB/FD CB prefix: indexed bit operations with register copy
  - Undocumented instructions: IXH, IXL, IYH, IYL, SLL (CB 30-37)

Accuracy:
  - Precise T-state counting for every instruction
  - Correct behavior of all flags including undocumented bits F3 (bit 3)
    and F5 (bit 5)
  - Internal MEMPTR/WZ register with correct influence on F3/F5
  - DAA with complete BCD correction logic (2048-entry lookup table)
  - LD A,I/R HW bug: INT after LD A,I or LD A,R resets PF to 0

Interrupt system:
  - IM0: vectored interrupt (instruction on data bus)
  - IM1: fixed jump to 0x0038
  - IM2: vectored interrupt (I:vector -> table)
  - NMI: non-maskable interrupt (jump to 0x0066), preserves IFF2
  - EI delay: interrupt is deferred by one instruction after EI
  - INTACK callback for daisy chain peripherals
  - RETI callback for peripheral notification
  - Level-sensitive INT line (polling callback)

Optimizations:
  - Computed goto dispatch (GCC/Clang)
  - Local register cache in z80_execute()
  - Null-check elimination for callbacks (default handlers)
  - DAA lookup table
  - Inline prefix handlers

API:
  - z80_init(), z80_reset()
  - z80_step(), z80_execute()
  - z80_irq(), z80_nmi()
  - Callback system for memory, I/O, interrupts
  - z80_add_wait_states() for extra wait states from callbacks
  - z80_set_post_step_fn() for per-instruction callback

Test suite:
  - 402 tests covering all instructions, flags, timing,
    interrupts, MEMPTR and undocumented behavior
  - ZEXALL validation: 67/67 Z80 Instruction Exerciser tests passed
    (including undocumented instructions and flags)
